LastPass

これまでのメイン。Firefox/Chorme/Opera/Safari/IEとひと通り対応していて、 Android/iOSでも使えます。

PC/Macだけならいくらでも無料で使えます。

モバイルが有料なので、お金を払ってモバイルと同期するかその他を考える必要があります。

PaswordBox

最近見つけたもの。使い勝手はLastPassと同じ感じです。 (Firefoxでインストールするとき、Firefoxのアドオンページにはないので、 直接ホームページ: http://j.pbox.io/jzmYwDFQ からダウンロードしてインストールする必要があります。)

LastPassとの違いは

  • 無料で保存できるのは25個まで
  • その代わり、モバイルも無料

ので、LastPassと優劣つけがたいところです。 保存してる数が25個以下ならPasswordBoxのほうがよさ気。

ので、しばらくこちらを使ってモバイルも同期してみようかな、と思っています。

とりあえずLastPassも付けた状態でパスワードを入力すると PasswordBoxの方にも記憶されるので、少しずつ移動中。

追記: 2015/05/15

  • これを入れると新しいタブで開いたページがPasswordBoxの物になってしまいます。

PasswordBox : I don’t want to use the StartPage, …: https://help.passwordbox.com/customer/portal/articles/1349933-i-don-t-want-to-use-the-startpage-can-i-turn-it-off-

ちょっとマルウェア的な動作で嫌ですが、 対処するためにはこのページを開いた状態で右下の歯車へ行って設定ページに飛べるので、 start pageを使うがOnになってるのでこれをOffにすれば解除されます。 (逆に一度オフにした後このページを開くすべがもはや分からない。。。) 新しいバージョンはこの様なことをしない、とのことですが、 新しいバージョンだと更に色々設定が追加されてますが いずれにしろこの設定になってしまいました。。。

追記: 2015/05/15

それから、25個まで、ということなんですが、25個まで埋めた時点で、 常にOff?の状態から開放できなくなってまともに使えなくなってます。

25個だけでも使える方法があるのか、25個になった時点で 有料版にしなければいけないのかよくわかりませんが、 あまりうまく使えない。。。

追記ここまで

1Password

1Password - Password Manager
デベロッパ: AgileBits Inc.
無料

巷でよく出てくる有料アプリです。

パスワードという重要事項を扱うものなので、多少高くてもきちんとしたものを 買いたい、ということで買ってる人も多いのでしょうか。。。

使い勝手は使ってる人によると良さそうですが。

KeePass

こちらはオープンソースのパスワードマネージャーです。

サーバーなどがあるわけではないのでローカルで動かすことが前提ですが、 自分でDropboxなんかでキーファイルを同期すれば他の環境との同期ができます。

元がオープンソースなので、その辺の関連ソフトみたいのも結構あるみたいです。

Firefoxの同期

Firefoxが29にアップデートされ、FirefoxSyncも刷新され、 Firefox Accountを使って他の同期サービスと同じような 同期が可能になっています。 Android版ももちろん同期できます。

なので、Firefoxだけであれば他にブックマーク、アドオンの同期も含めて 全部これでも良いかも。 (特にアドオンはこれでしかできないので)

ただ、パスワードの取り出しが他に比べて簡単にはできないかと。

これに統一すればブックマーク同期のXmarksすら要らなくなるわけですが、 やはり他との連携もしたいところ。

Chromeの同期

こちらも同様。こちらの方は前からモバイルも全部簡単に同期できるように なっていました。

これもFirefoxと同様、すべてChromeにすれば良いのだが、という話。

Firefoxと違ってiPhoneでも使えるため、ほんとにChrome使いになれば これだけでも大丈夫かもしれませんが、 やはりVimperatorがあるので どうしてもFirefoxを使いたいので。。。

ただし、iCloud3.0(今は3.1)はVista以前は使えず、 Windows７とWindows８のみ対応、とのことです。

まとめ

色々と付け過ぎるとconflictしてしまうので(特にブックマークの同期の方が) 気をつけないと行けませんが、 Firefoxも新しくなったことだし、 とりあえず、LastPassをメインにしてPasswordBoxも有効にし、 FirefoxSyncも一応動かしています(アドオンを同期したい)。

Xmarksもブックマーク同期として動かしていますが、今のところ変なconflictはない模様。

全部をずっと動かしてても意味が無いし、下手にパスワードマネージャーをたくさん使うのは 危険でしかないのでそのうち見極めようと思いますが、

  • PasswordBoxで十分ならPasswordBoxへLastPassから乗り換え。
  • FirefoxSyncの方はアドオンだけに。
  • ブックマークはXmarksで。

みたいな感じにしようかな、と今のところ思っています。
