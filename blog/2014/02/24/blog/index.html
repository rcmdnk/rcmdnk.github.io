はてなブックマーク数付きで貼り付け用リンクを作るブックマークレット

本題に入る前に、 せっかくなのでブックマーク数を付けたリンクを載せたかったので それ用のブックマークレットを作ってみました。

表示中のページのはてなブックマークページを開くブックマークレット

で書いたとおり、はてなブックマークのページは

http://b.hatena.ne.jp/entry/+<Page URL>

で行けるんですが、 さらにブックマーク数も

http://b.hatena.ne.jp/entry/image/+<Page URL>

することでそのページのブックマーク数を表す数字の画像が取得できます。

勘違い系

ローカルでテストしてたり設定用ページのURLだったりを間違えてブックマークしてしまったケース。

ブックマークのコメントで明らかに違うページを意図してる物があって面白い。

  • about:blank
    • よく見る人気ページ。ブックマークの数もすごい。
  • about:config
    • こちらは流石にブックマークする機会は少ないかな、と。
  • localhost
    • 自分のページをブックマークしたと思ったら良くわからないコメントが既に沢山付いてた、という。
    • トップページだけじゃなくてhttp://localhost/+~をブックマークしてる人は沢山居そう。。。その場合は他のコメントも無いだろうから気づかないし。。。
  • localhost:80
    • 通常Webは80番ポートなので上のlocalhostと同じなんですが、はてなブックマークだと微妙に区別される場合があるらしい。ただ、 このエントリーには、他のブックマークエントリーページが存在します。以下のリンク先から、別のコメントなどの情報も参照できます。 という言葉とともにlocalhostへのリンクがはられてる。 この区別されるのがある時期より前だけなのか、:80を入れても:80無しのページヘすぐにリダイレクトされるページがほとんど。
  • localhost:8080
  • localhost:8000
    • テストとかで使ったり。
  • 127.0.0.1/
    • localとは別物らしい。こちらにも8080, 8000番バージョンもあります。
  • 192.168.0.1/
  • 192.168.0.2/
    • うちで使ってるアドレスを使わないでください案件。
    • 192.168.0.1/の方には最後の/が無い192.168.0.1 バージョンが区別されて存在します。
    • この/についても:80の場合と同様、別れてるものとリダイレクトされる物があります。 無理やり作れば今でも出来るのでしょうか？

存在しないページを敢えてブックマーク

  • test
    • テスト用。。。？
    • こういったドメイン無しの様なURLの場合、通常ページを見に行こうとするとwww.test.comみたいなページにリダイレクトされたりします。
    • ただ、test.com と言うページにもぶくマがこの場合はあって、その様な場合にはhttP://test/だけを見ている様です。
  • a
    • なぜかはてなアンテナ。
  • a/
    • スラッシュ付き。こっちのが沢山。
  • aa/
    • a2個
  • aaa/
  • aaaa/
  • aaaaa/
  • aaaaaa/ はてブ
  • aaaaaaa/ はてブ
    • 調べた時点ではa6個まで(最後のは誰かが付けて消したのか0ブクマ)。7個目を最初にブクマしたければチャンスです。
  • hoge
  • hoge/
    • hogeも。
  • mac/
  • mac.com/
  • Apple - Mac
    • 上のmacは流石に何もなし。ただ、mac.comは3番目に挙げたAppleのMacのページへリダイレクトになるらしい。
  • windows/ はてブ
  • windows.com/ はてブ
  • Microsoft Windows - Microsoft Windows
    • windowsはまだ誰もブックマークしてない。2番めのページは日本語環境なら3番目のページヘのリダイレクト。

はてな関連

  • hatena
    • コメントが不明。(hatena.comの方へのコメントかな？)
  • はてな(www.hatena.com)
    • もともと英語圏用？
  • hatena.com
    • 上のwww.hatena.comと同じページ。
  • はてな(www.hatena.ne.jp/)
    • 本家。
  • はてな(hatena.ne.jp/)
    • www無し。(ページは一緒。)
  • はてなブログ
  • はてなブックマーク
    • 特に何と言うわけではなく各サービスのページ。

はてなブックマーク数ランキング

少し毛色が違いますが、ついでにはてなブックマークの総合ランキング についても見てみることに。

  • 人気エントリー - はてなブックマーク
    • はてなブックマーク数の総合ランキング。
    • 何故かYahooが13700超えと圧勝。1万を超えてるのはこれとはてなブックマークの登録ページ(はてなブックマーク - はてなブックマークをはじめる (セットアップ))だけ。
    • ここでもabout:blankが上位に食い込んでて人気ページであることが分かる。
