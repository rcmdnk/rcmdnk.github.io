家の固定回線の更新にあたってやっぱりWiMAXを使うことにした

  • B! 3

  • 0

  • 0

  • 0

  • 0

これまで家では2年前に契約したWiMAXを家でのネット環境として使っていました。

光などと違って工事とかがない分すぐに契約して使えるのと、 Try WiMAXとかをやってみて思ったよりも普通に使えたので、 最悪嫌になったら外持ち出し用で別に契約すればいいか、ということで契約して 結局固定回線は別に契約しないままここまで。

その契約が2年経って契約更新つきが来たので色々見てみましたが 結局WiMAX 2+を契約することにしました。

注

以下の記事は2016年時のもので キャンペーンのキャッシュバック金額など現在と異なる場合があります。

現在の詳細については各種ページでご確認ください。

これまでのWiMAX

これまではRacouponで契約したWiMAX (無印)を使っていました。

今は他も一緒ですが無印WiMAXの契約は無くてWiMAX 2+だけになっています。

ルーターは Mobile Slim。

すでにWiMAX 2+は当時サービスを始めていましたが、 Try WiMAXレンタル をしてみたところ、当時の家ではWiMAX 2+の電波は入ったり入らなかったり な感じだったので使えない可能性の中で千円か二千円高くなるのもな、 というのと、無印WiMAXでも普通に使う分には十分な速度だったので 普通のWiMAXに。

WiMAXに関しては基本的にプロバイダーは全く速度等には関係ないので 後は値段とサービスだけなんですが、 まあサービスは良い噂より悪いうわさばかりのWiMAX業界なので 値段だけ見てRacouponにしました。

キャッシュバックとかが無い代わりに一月2000円で当時はキャッシュバックとか含めても一番安かったので。 (その後500円/月のが出たりする祭りもありましたが。。。) 後、この契約の時には24ヶ月以内だと解約金が取られますが、 その後はいつ解約しても解約金無し、というのが良い所でした。 (ただし、25ヶ月目以降は割引がなくなって1000円以上高くなる。)

ルーターにはいくつか種類がありましたが、取り敢えず一番安いのでMobile Slimに。

2年間使ってみましたが、思ったよりも快適に使えました。 時々ものすごく遅くなるような時もありましたが支障を生む事はありませんでした。 動画とかもそれだけ見るだけならHDDのでも問題なく。

ということで、割りと快適に使えてたのでそのまま使い続けても良いくらいだったんですが、 値段が上がるし今更無印WiMAXを使い続けるのもあれなので別なものを探しに。

光 or ADSL

最近WiMAXに関しては悪いうわさしか聞かないので今度は 固定回線かな、という感じで探していました。

流石に前までの2000円とかではないものの、3000円位までを目処に。

探してみると、キャッシュバック含めると2000円ちょっとで出来そうなのを発見。

So-netのauひかりだとマンションタイプ2年間2300円/月で 4万円キャッシュバックとのこと。 ただここだとキャッシュバックの半分はソネットポイント。

キャッシュバック2万円と考えてもまあ2000円ちょっとなのでありかな、 と思ってたんですが、これ、使えるところが限られていてマンションごと契約とか してないとダメな模様。 ウチを調べたら入ってなかったので断念。。。

これの場合はAUなのでAUでスマホとか使ってると更に数百円から2千円位安くなったりするので AUを使ってて住んでるところが対応してたらまず契約したら、という感じ。

こっちだと新規なら3700円/月で240000円のキャッシュバック。

ただ、この辺色んな所でいろんな違うキャンペーンやってるので 下の価格.comとかだと5300円/月で56000円キャッシュバックとか。

光ファイバー・マンション プロバイダ比較（光ファイバー・ADSL） - 価格.com

それだと2年で考えると2800円/月程。

ただこっちだと3年目に入った時に上のと比べて大分高い感じです。 (その辺の料金体系がどうなってるのかもう見てもよくわからなすぎ。。。)

なんにしろ、探せばマンションなら3000円/月位で出来るのはありそうです。

ただ、見てみるとどれもやはり工事に時間がかかる可能性があって、最大一ヶ月程かかるとのこと。 今回に関して言えばWiMAXがある状態で余裕を持ってやれば大丈夫ではあるんですが、 新しく引っ越してすぐ使いたいときなんかは微妙です。

それから、キャッシュバック付きの物とかは大概工事費が月額料金に含まれていて(1500円程)、 解約すると24ヶ月分(もしくは30ヶ月契約の所も)の1500x残りを払わないといけなかったりします。 また、他に解約手数料が取られたり、早く解約するとそもそもキャンペーンも貰えなかったり。

特にAU光とかで契約してしまって次に住むところが対応してないと強制的に解約になって無駄な支払いをすることも。。。 最低2年なり30ヶ月使ったっとしても、契約更新付きは一ヶ月でまた2年契約とかになるので、 やはり引っ越しとかの可能性があるとかなりやっかいです。

などと考えてた所、なかなか固定回線にするのはどうなのかな、と思いとどまってしまいました。

ADSLも価格.comのところで見てみると、キャッシュバックの量は大きい物は少なく、 月額料金がもともと3000円位のものがあります。

ADSLタイプ2 プロバイダ比較（光ファイバー・ADSL） - 価格.com

素直に月額料金を見るとやはり安いですが、 2年でキャンペーン付きで考えると光もADSLも同じ様なもんなので、 それなら光で良いかな、と。

ただ、ADSLだと最低契約期間が1年とかで、それ以降はいつでも解約できる、と言ったものがあるので、 引っ越しの可能性が多いなら選択肢としてありかも。 (とは言っても引っ越してもAU光とかでなければよほどの場所か海外に行くとか出ないかぎりは そのまま移行出来る可能性は大きいと思いますが。)

WiMAX

今は無印のWiMAXのみでの契約は無くなって、全てWiMAX 2+での契約になっています。

ルーターとしては Speed Wi-Fi NEXT WX02 か Speed Wi-Fi NEXT W02 と言うのが今の状況で、 WX02の方はWiMAXにもつなげることが出来、 一方、W02の方はAUのLTEにつなげる事が出来ます。

WX02では通信を選べるわけではなく、WiMAXしか無いところでWiMAXに繋がるようですが、 今ではWiMAXの部分をどんどんWiMAX 2+に変更している状態でWiMAXのみ、みたいな所のが レアな状態かもしれません。

一方、W02の方ではLTEにハイスピードプラスエリアモードとしてつなげる事が出来ますが、 こちらを使うと1000円/月程余計にかかります。

なので敢えて携帯使わずLTEを使う必要も無いのでWX02の方がよさ気。 ディスプレイに表示できる情報とかもWX02とかだと 過去3日分のデータ量とか色々出来ます。

後はWiMAX 2+から月間容量が無限でなくなったのでそこを注意しないといけません。

通常プランは月間7GBで超えると128kbpsに制限されます。 これはLTEを使った場合でも合計で計算されます。

一方、悪名高き ギガ放題ですが、月間容量には制限が無いものの、 3日間で3GB以上使うと制限がかかります。

最初はこの制限で128kbpsとか現状のパソコン作業で使い物にならない位の速度だったりしたので ちょっとしたいざこざっぽい状態になってましたが、 その後改善され、128kbpsとかではなく、Youtube動画の標準画質が見れる以上の 速度になっている、とのこと。

UQ FlatツープラスUQWiMAX 超高速モバイルインターネットWiMAX2+

具体的な数字は出ていませんが、実際使っている人の話なんかを見ていると 5Mbpsとか位は出ている様です。

実測でそれだけ出てれば無印WiMAXよりは快適に使える状態なので取り敢えずは使える状態だと思います。

ということで、ギガ放題を固定回線にするのはありえない、と言った話を たくさん見ていたので流石にWiMAXで今後も行くのはどうかな？と思ってましたが、 どうも制限がかかった状態でも結構使えるっぽいので良いんじゃないか、と。

ということで、後は値段。

こちらも色々プロバイダーがあって、かつ、キャンペーンが同じプロバイダーでも 入るとこ違うと違ってたりめちゃめちゃですが、 サービス内容としては通常プランかギガ放題か、と言ったところです。 後は上にも書いたルーターの種類。

実際どれだけ使うのか、と言うことで、Windows 10で調べてみました。 Windows 10では設定ネットワークとインターネット データ使用状況で 過去30日のデータ使用状況が見れます。 見てみたところ、60GB程度。。。ということで1日平均2GBで、 3日で3GBは3日も持たずに超えてしまいそうです。 1月7GBの制限は問題外。

ということで、常に制限付き覚悟でギガ放題で契約する、という選択肢しかありません。

ということでギガ放題の物をいくつか見てみましたが、 良く出てくる安いやつというのが 上のとくとくBBのもの。

ただ、上のページに行くと ギガ放題は月3414円/月でキャッシュバックとかが無いプランです。

とくとくBBのページにいくと ギガ放題4263円/月でキャッシュバック30000円、と言うのがあります。

2年契約したとすると、月1250円割引なので30000円/月くらいになります。

(どちらも最初の2ヶ月間ギガ放題分の700円位の割引があります。)

ただし、とくとくBBのキャッシュバックは検索すると出てきますが やたら評判が悪く、ちょっと見過ごしていただけで取り逸れる事もあるみたいなので 一気にキャッシュバックが来るわけではないけどそこそこ安い上のものもまあ良いのかも。 キャッシュバックは1年後位に来るのでそれまでは3414円のプランの方が支払いは少なくなります。

後は、これまはRacouponのWiMAXを使ってきたんですが、 いま見てみたら古いルーターを使った安いプランがあります。

W01というW02の前の機種ですが、これだと月2844円とのこと。 さらに提供月から3ヶ月目までギガ放題無料。

WX02のプランだと3664円になっています。

どちらも楽天クーポン1000円分付き。

ちょうど決めた時はこのW01のプランは無かったのですが。。。

WX02のプランもあって、こちらは3664円/月ととくとくBBのキャッシュバックなしの物とほぼ同じ。 楽天クーポン1000円分が別途ついてきます。

以前はこれとは別に今WiMAX契約してる人用に乗り換えでWX02にするプランがあって、 こちらは2950円/月でした。 このプランは上のプラント違って安心サポートが付いていないものです。 (故障時に修理とかが無料になるとか。)

ただ、もともとメーカー保証が1年付いてたりするのでそこまで必要ないとは思いますが、 この費用分が700円となってるようです。

ということで、WiMAx 2+なら大体月3千円位で使えそうだということに。 これだと光と同じくらいになりますが、 速度や安定性を取るか、導入までの時間や引っ越しなどの時の面倒さ、 さらには携帯WiFiとして外に持ち出せるかどうか、とかを考えると WiMAX 2+かな、ということになりました。

確実に3日3GB制限にひっかかるので、制限に引っかかった時にどれだけ遅くなるか、という懸念がありましたが、 どうやら前まで使ってたWiMAX以上には速度が出るということで 全然問題ないかな、という結論。

後はプロバイダーをどこにするかですが、 これまでのRacouponだったのでそのまま乗り換えても良いのですが、 全部含めると微妙にとくとくBBのが安くなるのと、 せっかくなので違うところにしよう(実際に使い始めれば何も関係ないのですが。。。)、 ということでとくとくBBのキャッシュバックをもらうものにしました。

とくとくBBのキャンペーについては、申し込んだ時は クレードルのあり、なしを選べてなしにすると32000円のキャッシュバックでした。 (3月くらいには37000円くらいだったりもしたらしい。。。) こちらの方だと初期費用の3000円から1000円割引があったりするので それらも入れると結局月平均3000円位になります。

ただ、実際の毎月の支払いは4300円程度。 後はキャッシュバックを忘れずに。。。

追記: 2017/03/31

キャッシュバックの連絡が来たので手続きをトリました。

  • とくとくBBのWiMAXのキャッシュバックの連絡が来たので手続きを行った(のとRacoupon WiMAXの話)

追記ここまで

WiMAXにしたもう一つの大きな理由として、 この前携帯をFREETELのSIMに変更して、月額がデータ使用量に従って上がる契約なので、 できればテザリングをあまりしたくありません。 WiMAXがあれば他のネットが使えない所でもテザリングの代わりに使うことが出来るのでいいな、と。

  • 携帯の更新月が来たのでFREETELに乗り換えた

これまでもWiMAXを持って行って便利なことが多かったので。 WiMAX 2+になって、下に書く通り、近場でも電波がかなり良くなってるので 他の所でも良くなってるだろうと思うので外でも使えるようになるだろうと。

Tr WiMAX

本来であれば、Try WiMAXとかでもう一度ウチでのWiMAX 2+の通信の状況や 制限がかかった時にどれ位になるか、というのを確かめた方が良かったのですが、 Try WiMAXをやるにはちょっと面倒が。

Try WiMAXレンタル

上のページから申し込めるんですが、申し込むには大手キャリア(Docomoとか)の メールアドレスが必要です。 Gmailではダメ。 おそらく無駄に同じ人が繰り返し申し込めないようにするためとかの措置なんでしょうが、 流石に今の時代にキャリアのメアドが必要というのはちょっと驚きです。

キャリアアドレスがない場合には書面で送ることも出来ますが、まあ面倒です。。。 前は親が古い携帯でアドレスあるのでそれを使いましたが今回は面倒さを考えると まあ大丈夫かな、ということでやらずに直接申し込みました。

WiMAX 2+ (WX02), とくとくBB

ということでWiMAX 2+を申し込みましたが、 ウェブで申し込み後、ルーターは3日後に届いてすぐに使いはじめることが出来ました。

2年ほど前はWiMAX 2+の電波をほとんど捉える事が出来ませんでしたが、 今回は常にアンテナ4本中4本立ってる状態なのでひとまず安心。

取り敢えず制限がかかってない状態で速度を測ってみると、

BNR スピードテスト 回線速度／通信速度 測定

このサイトで

  • WiMAX 2+ (WX02) USBで有線として繋いだ場合: 下り9Mbps, 上り 3Mbps
  • WiMAX 2+ (WX02) WiFi (2.4GHz) : 下り2Mbps, 上り 2Mbps
  • WiMAX (Mobile Slim) WiFi (2.4GHz) : 下り700kbps, 上り700bps

なんかちょっと遅い?

別のサイトでも調べてみました。

Speedtest.net by Ookla - The Global Broadband Speed Test

こちらだと

  • WiMAX 2+ (WX02) USBで有線として繋いだ場合: 下り30Mbps, 上り 5Mbps
  • WiMAX 2+ (WX02) WiFi (2.4GHz) : 下り15Mbps, 上り 4Mbps
  • WiMAX (Mobile Slim) WiFi (2.4GHz) : 下り 5Mbps, 上り 2Mbps

といった感じでそこそこ速い感じです。 測定サイト側との距離とか間の問題もあるので難しいですが、 まあ速い経路なら下の様な速度くらいは出るのかな、と思って良いと思います。

  • Androidスマホでの3つのテザリングの仕方。設定のやり方や速度を比較等。

前に上のとこで書いた様にスマホのテザリングも含めてWiMAXの速度を調べたりもしましたが、 この時は上の方のサイトで結果も同じ様なものでした。 今回も一応何度か調べてみて大体上みたいな感じになってました。 時間帯にもある程度はよると思いますが前の結果と同じくらいなのでだいたい それぞれの結果自体はそんなものなのかな、という感じ。

体感的にはWiMAXも700bpsということは無いと思うので 経路が悪くなければ下くらいの速度は出ると考えて良いんだと思います。

また、申し込むときは考えてませんでしたが、 有線で繋ぐと結構速度が違います。

この3日後、当然すぐに3GBの制限を超えて速度制限に引っかかったわけですが、 引っかかった後の測定をすると

BNR スピードテスト 回線速度／通信速度 測定

  • WiMAX 2+ (WX02) USBで有線として繋いだ場合: 下り5Mbps, 上り 5Mbps
  • WiMAX 2+ (WX02) WiFi (2.4GHz) : 下り2Mbps, 上り 2Mbps
  • WiMAX (Mobile Slim) WiFi (2.4GHz) : 下り1Mbps, 上り 2Mbps

Speedtest.net by Ookla - The Global Broadband Speed Test

  • WiMAX 2+ (WX02) USBで有線として繋いだ場合: 下り5Mbps, 上り 5Mbps
  • WiMAX 2+ (WX02) WiFi (2.4GHz) : 下り5Mbps, 上り 4Mbps
  • WiMAX (Mobile Slim) WiFi (2.4GHz) : 下り4Mbps, 上り 3Mbps

こんな感じ。 有線で繋いだ時に明らかに遅くなっていました。 ただ、それでも下り5Mbps程は保っているし WiFiの状況に関して言えばそれ程下がっていません。

後、BNRスピードテストの方だと下がり度合いが少ないです。

テストの下がり度合いが少ないのはそもそもそれ程高速な通信が無理な経路になってるからかな、と。

WiFiでの下がり具合が少ないのも、WiFiにすることで制限される速度が 5Mbps程度まで、ということだと思います。 (ただ、それ以上のそくどが普通に出てる感じのをよく見るので実際の所どうなんでしょう。。。？)

いずれにしろ速度制限はかかりましたがそれでも普通に使えるし、 使っている感じではそれ程速度が変わったのかわからないくらいです。

まとめ

家のネット環境の更新をどうしようかと思ってましたが、 結局引き続きWiMAXにしました。

WiMAX 2+のギガ放題が出てから制限に関して結構ひどい話が出ていましたが、 制限がかかってもよほどのことをして無い限りは普通に使えますし、 数年前のネット環境を考えたら十分速いと思います。

始めるにもすぐに使えるし、引っ越しなどの時にも手続きも全く無いので 取り敢えず楽なのは良いことです。
