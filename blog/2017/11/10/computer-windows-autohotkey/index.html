XсђЂYсЂ»тЄ║уЈЙСйЇуй«сЂД CoordMode сЂ«ToolTipсЂ«тђцсЂФсѓѕсЂБсЂдсѓ╣сѓ┐сЃ╝сЃѕтю░уѓ╣сЂїтцЅсѓЈсѓісЂЙсЂЎсЂїсђЂ тѕЮТюЪтђцсЂДсЂ»WindowсЂФсЂфсЂБсЂдсЂёсЂдсђЂ 0, 0сЂесЂЎсѓїсЂ░уЈЙтюесЂ«сѓдсѓБсЃ│сЃЅсѓдсЂ«тидСИісЂФсЂ▓сЂБсЂцсЂЈсѓѕсЂєсЂфУАеуц║сЂФсЂфсѓісЂЙсЂЎсђѓ

WhichToolTipсЂ»ToolTipсЂФТї»сѓЅсѓїсѓІуЋфтЈисЂДсђЂ1сЂІсѓЅ20сѓњТїЄт«џсЂДсЂЇсЂЙсЂЎсђѓ уюЂуЋЦсЂЎсѓІсЂе1уЋфсЂФсЂфсѓісЂЙсЂЎсђѓУцЄТЋ░сЂ«ToolTipсѓњУАеуц║сЂЋсЂЏсЂЪсЂёта┤тљѕсЂФсЂ» жЂЋсЂєуЋфтЈисѓњТїЄт«џсЂЌсЂЙсЂЎсђѓ(сЂЮсЂєсЂЌсЂфсЂёсЂетЅЇсЂ«сѓѓсЂ«сѓњСИіТЏИсЂЇсЂЌсЂдсЂЌсЂЙсЂёсЂЙсЂЎсђѓ)

сѓГсЃ╝тЁЦтіЏсѓцсЃЎсЃ│сЃѕсЂДУАеуц║сЂЋсЂЏсѓІу░АтЇўсЂфСЙІ

у░АтЇўсЂФсђЂAсѓњТі╝сЂЎсЂе1уДњсЂасЂЉToolTipсѓњтЄ║сЂЎсѓёсѓіТќ╣сЂесЂЌсЂдсЂ»

сЂЊсѓЊсЂфТќ╣Т│ЋсђѓToolTipсѓњтЄ║сЂЌсЂдсђЂ Sleep сЂДСИђуДњтЙЁсЂБсЂдсЂІсѓЅTextуёАсЂЌсЂ«ToolTipсѓ│сЃъсЃ│сЃЅсЂДтЅіжЎцсЂЌсЂЙсЂЎсђѓ

SleepсЂФСИјсЂѕсѓІт╝ЋТЋ░сЂ»сЃЪсЃфуДњсЂфсЂ«сЂД1000сЂДСИђуДњсЂДсЂЎсђѓ

ТгАсЂФУцЄТЋ░сЂ«ToolTipсѓњтЄ║сЂЌсЂдсЂ┐сЂЙсЂЎсђѓ

сЂЊсѓїсѓњт«ЪУАїсЂЌсЂдAсѓњТі╝сЂЎсЂесђЂ tool tip 2сЂеtool tip 3сЂїтЄ║сЂЙсЂЎсЂїсђЂ ToolTipуЋфтЈи1сЂ»tool tip 2сЂФСИіТЏИсЂЇсЂЋсѓїсѓІсЂ«сЂД tool tip 1сЂ»тЄ║сЂЙсЂЏсѓЊсђѓ

сЂЮсЂ«тЙїсђЂ1уДњсЂЪсЂБсЂдtool tip 2сЂїТХѕсЂѕсђЂсЂЋсѓЅсЂФ1уДњухїсЂБсЂдсЂІсѓЅtool tip 3сЂїТХѕсЂѕсЂЙсЂЎсђѓ

сЂЪсЂасЂЌсђЂсЂЊсЂ«SleepсѓњСй┐сЂБсЂЪТќ╣Т│ЋсЂ»сЂАсѓЄсЂБсЂетЋЈжАїсЂїсЂѓсЂБсЂдсђЂ 1уДњухїсЂцтЅЇсЂФAсѓњТі╝сЂЌсЂЪта┤тљѕсђЂ тєЇсЂ│tool tip 2сЂїУАеуц║сЂЋсѓїсѓІтЉйС╗цсЂїтЄ║сЂдсЂЊсѓїсЂ»сЂЮсЂ«сЂЙсЂЙУАеуц║сЂЋсѓїуХџсЂЉсѓІсЂЙсЂЙсЂДсЂЎсЂїсђЂ С╗ЦтЅЇсЂ«SleepтЙїсЂ«ToolTipсЂїућЪсЂЇсЂдсЂёсѓІсЂ«сЂДтЅЇсЂФAсѓњТі╝сЂЌсЂЪсЂесЂЇсЂІсѓЅ 1уДњсЂ«сѓ┐сѓцсЃЪсЃ│сѓ░сЂДТХѕсЂѕсЂдсЂЌсЂЙсЂёсЂЙсЂЎсђѓ

сЂЊсЂ«ТДўсЂфС║ІсѓњжЂ┐сЂЉсѓІсЂЪсѓЂсђЂжђџтИИсЂ»ТгАсЂ«SetTimerсѓњСй┐сЂБсЂЪТќ╣Т│ЋсѓњСй┐сЂёсЂЙсЂЎсђѓ

SetTimerсѓњСй┐сЂБсЂЪСЙІ

SetTimer сѓњСй┐сЂєсЂеТїЄт«џТЎѓжќЊсЂ«жђћСИГсЂДТХѕсЂЌсЂЪсѓісЂЎсѓІсЂЊсЂесЂїтЈ»УЃйсЂФсЂфсѓісЂЙсЂЎсђѓ

AсЂДToolTipсѓњтЄ║сЂЌсЂЪтЙїсђЂSetTimerсѓњСй┐сЂБсЂдсЂёсЂЙсЂЎсђѓ SetTimerсЂДсЂ»ТюђтѕЮсЂ«т╝ЋТЋ░сЂДсЃЕсЃЎсЃФсѓњТїЄт«џсЂЌсђЂ2сЂцсѓЂсЂФТЋ░тГЌсЂїсЂѓсѓІсЂе сЂЮсЂ«ТЎѓжќЊ(сЃЪсЃфуДњ)сЂ«сѓ┐сѓцсЃъсЃ╝сѓњСйютІЋсЂЌсђЂсЂЮсЂ«ТЎѓжќЊтѕєтЙЁсЂБсЂЪтЙїсЂФсЃЕсЃЎсЃФсЂДТїЄт«џсЂЋсѓїсЂЪтІЋСйюсѓњт«ЪУАїсЂЌсЂЙсЂЎсђѓ т«ЪУАїтЙїсЂ»сѓ┐сѓцсЃъсЃ╝сѓњсЃфсѓ╗сЃЃсЃѕсЂЌсђЂсЂЙсЂЪсѓФсѓдсЃ│сЃѕсЃђсѓдсЃ│сѓњтѕЮсѓЂсЂдТїЄт«џТЎѓжќЊтЙїсЂФтІЋСйюсѓњт«ЪУАїсЂЎсѓІсђЂсѓњу╣░сѓіУ┐ћсЂЌсЂЙсЂЎсђѓ

RemoveToolTipсЂесЂёсЂєсЃЕсЃЎсЃФсЂ«СИГсЂДсЂ» TextуёАсЂЌсЂ«ToolTipсЂДтЅіжЎцсЂЌсЂдсЂёсЂЙсЂЎсђѓ ТюђтѕЮсЂ«сѓ┐сѓцсЃъсЃ╝сЂї10000сЃЪсЃфуДњ(10уДњ)сЂДСйютІЋсЂЎсѓІсЂ«сЂДсЂЊсЂЊсЂДсЂ»сЃёсЃ╝сЃФсЃЂсЃЃсЃЌсЂї10уДњжќЊУАеуц║сЂЋсѓїсѓІсЂЊсЂесЂФсЂфсѓісЂЙсЂЎсђѓ

сЂЙсЂЪсђЂУАеуц║СИГсЂФAсѓњсѓѓсЂєСИђт║дТі╝сЂЎсЂе ToolTipсЂїсѓѓсЂєСИђт║дт«ЪУАїсЂЋсѓїсЂЙсЂЎсЂїсЂЊсЂ«жџЏсЂ»тљїсЂўсЃАсЃЃсѓ╗сЃ╝сѓИсЂфсЂ«сЂДсЂЮсЂ«сЂЙсЂЙсђѓ сЂЮсЂ«тЙїSetTimerсЂФсѓѕсЂБсЂдсѓ┐сѓцсЃъсЃ╝сЂїсЃфсѓ╗сЃЃсЃѕсЂЋсѓїсѓІсЂ«сЂД SleepсѓњСй┐сЂБсЂЪсЂесЂЇсЂежЂЋсЂёсђЂТюђтЙїсЂФТі╝сЂЌсЂЪсЂесЂЇсЂІсѓЅ10уДњухїсЂБсЂдсЂІсѓЅТХѕсЂѕсѓІсЂЊсЂесЂФсЂфсѓісЂЙсЂЎсђѓ

RemoveToolTipсЂДсЂ»сЂЋсѓЅсЂФToolTipсЂ«уЏ┤тЅЇсЂДSetTimerсЂДOffсѓњсЂЌсЂдсЂёсЂЙсЂЎсђѓ 2сЂцуЏ«сЂФOffсЂїсЂѓсѓІсЂесЂЮсЂ«сѓ┐сѓцсЃъсЃ╝сЂ»тЂюТГбсЂЋсѓїсѓІсЂ«сЂД сЂЊсЂЊсЂИТЮЦсѓІсЂесѓ┐сѓцсЃъсЃ╝сЂ»ТГбсЂЙсѓісђЂсЂЮсЂ«тЙїу╣░сѓіУ┐ћсЂЌсЂЙсЂЏсѓЊсђѓ ТюђтѕЮсЂ«SetTimerсЂІсѓЅсЃЕсЃЎсЃФсѓњС╝ЮсЂБсЂдт┐ЁсЂџТЮЦсѓІсѓѕсЂєсЂФсЂфсЂБсЂдсЂёсѓІсЂ«сЂД сЂЊсЂ«сѓ┐сѓцсЃъсЃ╝сЂ»СИђтЏъсЂЌсЂІСйютІЋсЂЌсЂЙсЂЏсѓЊсђѓ

сЂЊсЂ«УеГт«џсѓњсЂЊсЂ«ТДўсЂФтѕЦсЂ«сЃЕсЃЎсЃФсЂДСйюсЂБсЂдсЂісЂЈсЂеBсЂДсЂЊсЂ«сЃЕсЃЎсЃФсѓњтЉ╝тЄ║сЂЌсђЂ жђћСИГсЂДТХѕсЂЎсЂЊсЂесЂїтЄ║ТЮЦсЂЙсЂЎсђѓ

Gosub сЂ»сЃЕсЃЎсЃФсЂФсѓИсЃБсЃ│сЃЌсЂЌсЂдReturnсЂДТѕ╗сЂБсЂдсЂЈсѓІТЕЪУЃйсЂДсЂЎсђѓ Goto сЂесЂёсЂєС╝╝сЂЪсѓѕсЂєсЂфТЕЪУЃйсЂїсЂѓсѓісЂЙсЂЎсЂїсђЂсЂЊсЂАсѓЅсЂ»УАїсЂБсЂЪсЂЙсЂЙТѕ╗сЂБсЂдсЂЇсЂЙсЂЏсѓЊсђѓ сЂЊсЂЊсЂДсЂ»уЏ┤тЙїсЂФReturnсЂЌсЂдсѓІсЂ«сЂДсЂЕсЂАсѓЅсЂДсѓѓтљїсЂўсЂДсЂЎсЂїсђЂ GotoуџёсЂфсѓѓсЂ«сЂ»ТииС╣▒сЂ«тЁЃсЂфсЂ«сЂДGosubсЂДжќбТЋ░уџёсЂФтЉ╝сЂ│тЄ║сЂЌсЂдТѕ╗сЂБсЂдсЂЈсѓІсђЂ сЂеУеђсЂБсЂЪТёЪсЂўсЂФсЂЌсЂЪТќ╣сЂїтѕєсЂІсѓісѓёсЂЎсЂЈсЂфсѓісЂЙсЂЎсђѓ

сЂЊсЂЊсЂДсђЂBсЂФт»ЙсЂЌсЂдтЇўсЂФ

сЂ«ТДўсЂФToolTipсѓњуЏ┤ТјЦТХѕсЂЎТДўсЂФт«џуЙЕсЂЌсЂдсѓѓтЁесЂЈтљїсЂўсЂЊсЂесЂїтЄ║ТЮЦсЂЙсЂЎсђѓ сЂЪсЂасЂЊсЂ«та┤тљѕсЂ»УБЈсЂДсѓ┐сѓцсЃъсЃ╝сЂ»тІЋсЂЇуХџсЂЉсЂдсЂёсЂд3уДњтЙїсЂФToolTipсѓњт«ЪУАїсЂЌсЂЙсЂЎсђѓ т«ЪУАїсЂЌсЂдсѓѓTextсЂїуёАсЂёсЂ«сЂДухљт▒ђСйЋсѓѓУхисЂЊсѓісЂЙсЂЏсѓЊсЂїсђѓ

сЂЊсѓїсЂасЂЉсЂ«СЙІсЂасЂесЂЮсЂЊсЂЙсЂДТ░ЌсЂФсЂЎсѓІт┐ЁУдЂсЂ»сЂѓсѓісЂЙсЂЏсѓЊсЂїсђЂ СИјсЂѕсЂЪсЃЕсЃЎсЃФсЂ«СИГсЂДУЅ▓сђЁсѓёсѓІта┤тљѕсЂФсђЂУцЄТЋ░тЏъУхитІЋсЂЋсѓїсѓІсЂетЏ░сѓІсЂЊсЂесѓѓсЂѓсѓІсЂДсЂЌсѓЄсЂєсЂЌсђЂ тЇўсЂФУБЈсЂДуёАжДёсЂфсЃЌсЃГсѓ╗сѓ╣сЂїТ«ІсЂБсЂдсЂёсѓІсЂ«сѓѓТ░ЌТїЂсЂАТѓфсЂёсЂ«сЂД сЂЇсЂАсѓЊсЂеТХѕсЂЎсѓѕсЂєсЂФсЂЌсЂЪТќ╣сЂїУЅ»сЂёсЂДсЂЎсђѓ

тљїТДўсЂФсђЂSleepсѓњСй┐сЂБсЂдсѓІта┤тљѕсЂДсѓѓBсЂФToolTipсѓњт«ЪУАїсЂЋсЂЏсЂд уёАуљєсѓёсѓіТХѕсЂЎсЂЊсЂесѓѓтЄ║ТЮЦсЂЙсЂЎсЂїсђЂ сЂЊсЂ«та┤тљѕсѓѓSleepтЙїсЂ«т«ЪУАїтЉйС╗цсЂ»Т«ІсѓісЂЙсЂЎсЂЌсђЂ СИісЂФсѓѓТЏИсЂёсЂЪсѓѕсЂєсЂФSleepсЂДсЂ«т«ЪУАїсЂФсЂ»тЋЈжАїсЂїсЂѓсѓІсЂ«сЂД SetTimerсѓњСй┐сЂБсЂЪТќ╣сЂїУЅ»сЂёсЂДсЂЎсђѓ

GuiсЂ«сѓфсЃќсѓИсѓДсѓ»сЃѕсЂФсЃъсѓдсѓ╣сѓфсЃ╝сЃљсЃ╝сЂДToolTipсѓњУАеуц║сЂЋсЂЏсѓІ

TooltipсѓњуЏ┤ТјЦУАеуц║сЂЋсЂЏсѓІсЂЊсЂесЂ»тЄ║ТЮЦсЂЙсЂЌсЂЪсЂїсђЂ WebсѓёС╗ќсЂ«сѓбсЃЌсЃфсЂДУЅ»сЂЈсЂѓсѓІсЂ«сЂ»СйЋсѓЅсЂІсЂ«сѓѓсЂ«сЂ«СИісЂФсЃъсѓдсѓ╣сѓњТїЂсЂБсЂдсЂёсЂБсЂЪТЎѓсЂФ УАеуц║сЂЋсѓїсѓІToolTipсЂДсЂЎсђѓ

сЂЪсЂасЂЌсђЂTextсЂ»сЂАсѓЄсЂБсЂеТ│еТёЈсЂїт┐ЁУдЂсЂД сЂЮсЂ«сЂЙсЂЙv-labelсѓњСИјсЂѕсЂдсѓѓсЃєсѓГсѓ╣сЃѕсѓњСИјсЂѕсЂдсѓѓGui ControlсЂДсЂ»сЂфсЂёсЂ«сЂД A_GuiControlсЂФСйЋсѓѓтЁЦсѓісЂЙсЂЏсѓЊсђѓ (СИісЂ«тЈѓУђЃURLсЂ«ТюђтЙїсЂ«У│фтЋЈсЂФсЂЊсѓїсЂФжќбсЂЎсѓІсЂеТђЮсѓЈсѓїсѓІсѓѓсЂ«сѓѓсЂѓсѓісЂЙсЂЎсЂїсђѓ)

сѓ│сЃ│сЃѕсЃГсЃ╝сЃФсѓфсЃќсѓИсѓДсѓ»сЃѕсЂФсЂЎсѓІсЂЪсѓЂсЂФсЂ»g-labelсѓњС╗ўсЂЉсЂЙсЂЎсђѓ сЂЊсѓїсЂДТі╝сЂЌсЂЪТЎѓсЂФсЂЎсѓІтІЋСйюсѓњС╗ўсЂЉсѓЅсѓїсѓІсѓЈсЂЉсЂДсЂЎсЂїсђЂ С╗ітЏъсЂ»сЂЊсѓїсЂФСйЋсѓѓсЂЌсЂфсЂётІЋСйюсѓњDummyсЃЕсЃЎсЃФсѓњтЅ▓сѓітйЊсЂдсЂдсЂёсЂЙсЂЎсђЂ сЂЊсѓїсЂФсѓѕсѓісђЂТі╝сЂЌсЂдсѓѓСйЋсѓѓсЂЌсЂфсЂёсЃєсѓГсѓ╣сЃѕсЂДсЂЎсЂї Gui ControlсЂесЂфсѓісђЂA_GuiControlсЂДтђцсЂїтЈќтЙЌсЂДсЂЇсѓІТДўсЂФсЂфсѓісЂЙсЂЎсђѓ

тђцсѓњтЈќтЙЌсЂЌсЂЪсѓЅсђЂтЅЇтЏъсЂетцЅсѓЈсЂБсЂдсЂфсЂЉсѓїсЂ░СйЋсѓѓсЂЏсЂџсЂФухѓС║єсђѓ

СИісЂ«URLсЂ«СЙІсЂасЂе!=сЂ«С╗БсѓЈсѓісЂФ< >сѓњСй┐сЂБсЂдсЂёсЂЙсЂЎсЂїтљїсЂўТёЈтЉ│сЂДсЂЎсђѓ сЂЙсЂЪсђЂInStr(...)сЂД" "сЂеТ»ћсЂ╣сђЂуЕ║уЎйсѓњтљФсЂЙсЂфсЂёсЃЂсѓДсЃЃсѓ»сѓњсЂЌсЂдсЂёсЂЙсЂЎсЂїсђЂ сЂЊсѓїсЂ»уЕ║уЎйсѓњтљФсѓђсЂесђЂсЂѓсЂесЂД

сЂ«ТЅђсЂДсђЂ%CurrControl%сЂ«тђцсЂе_TTсѓњсЂЈсЂБсЂцсЂЉсЂЪтљЇтЅЇсЂ«тцЅТЋ░сЂ«СИГУ║ФсѓњтЈќсѓітЄ║сЂЎТЎѓсЂФ тцЅТЋ░тљЇсЂФуЕ║уЎйсЂїсЂѓсѓІсЂесѓесЃЕсЃ╝сЂїтЄ║сЂдсЂЌсЂЙсЂєсЂІсѓЅсЂДсЂЎсђѓ

сЂЪсЂасђЂсЂЮсЂ«ТДўсЂфтѕЦсЂ«УдЂу┤асЂФуД╗сЂБсЂЪжџЏсЂФсѓѓTooltipсѓњТХѕсЂЌсЂдсЃфсѓ╗сЃЃсЃѕсЂЎсѓІсЂ«сЂїТГБсЂЌсЂёсЂ«сЂД тцЅТЏ┤тЙїсЂ»сЂЙсЂџA_GuiControlсЂїтцЅсѓЈсЂБсЂЪсЂІсЂЕсЂєсЂІсЂасЂЉУдІсЂдPrevControlсѓњсѓ╗сЃЃсЃѕсЂЌ ToolTipсЂДСИђт║дToolTipсѓњтЅіжЎцсЂЌсЂдсЂёсЂЙсЂЎсђѓ

сЂЊсЂ«жЃетѕєсЂФжќбсЂЌсЂдсѓѓсЂєт░ЉсЂЌсђѓтЈѓУђЃURLсЂ«Тќ╣сЂДсЂ» Рђдand not InStr(CurrControl, " ")сЂ┐сЂЪсЂёсЂфсЂЊсЂесѓњсЂЌсЂдсЂёсЂЙсЂЎсЂїсЂЊсѓїсЂ»СИіТЅІсЂЈтІЋсЂЇсЂЙсЂЏсѓЊсЂДсЂЌсЂЪсђѓ
