システム環境設定からの設定

コマンドラインから一発で設定する様なものを作りたいとは思ってるんですが、 ちょくちょく設定項目が変わったりもするので 今回も一度一通り見ながら設定してみました。

  • Macでdefaultsコマンドで設定出来るオプションを調べる

一般

  • アピアランス：グラファイト
  • メニューバーとDockを暗くする: チェック
  • 強調表示色: グラファイト

デスクトップとスクリーンセーバー

  • デスクトップ: 無地の色右真ん中にある黒目の灰色。
  • スクリーンセーバー: 開始までの時間: 開始しない

Dock

  • サイズ: 小さめに
  • 書類を開く時はタブで開く: 常に
  • Dockを自動的に表示/非表示: チェック

Mission Control

  • 最新の仕様状況に基づいて操作スペースを自動的に並べ替える: チェックを外す。
  • ウインドウをアプリケーションごとにグループ化: チェックを外す。
  • ディスプレイ毎に個別の操作スペース: チェックを外す

ディスプレイごとに個別の操作スペース のチェックを外すと El Capitanから導入されたSplit Viewは無効になります。

  • OS X El CapitanでSplit Viewを有効化する方法

個別の操作スペースを有効にするとすべての画面で目にィーバーが出たりしますが、 むしろ画面が小さくなるだけで邪魔なので無効にしたいのとSplit Viewは特に使わないのでチェックは外します。

言語と地域

  • 優先する言語: English (英語)を加える。(日本語を最優先、だがこれ追加する意味ある?)

セキュリティーとプライバシー

  • ファイアウォール: オンにする

ディスプレイ

  • 解像度: “変更”にし、スペースを拡大、を選択。

省エネルギー

  • 電源アダプタ
    • ディスプレイがオフのときにコンピュータを自動でスリープさせない: チェック
    • 電源アダプタに接続中にPower Napをオンにする: チェックをはずす

キーボード

  • キーボード
    • キーのリピート: 最速
    • リピート入力認識までの時間: 最短
    • Control Stripをカスタマイズ: Touch Barでよく使うものとか。右側のを全部一度消して、画面ロック、明るさ調整、音量調整、消音をを追加。
  • ユーザー辞書
    • 辞書に関してはiCloud経由で同期されてる。(iPhone用の設定が主。あでメールアドレスを呼び出したり。)
    • 英字入力中にスペルを自動変換: チェックを外す
    • 文頭を自動的に大文字にする: チェックを外す
    • スペースバーを2回押してピリオドを入力: チェックを外す
    • Touch Barに入力候補を表示: チェックを外す
    • スマート引用符とスマートダッシュを使用: チェックを外す

追記ここまで

  • 入力ソース: Google日本語入力の英数とひらがなを追加しデフォルトの日本語を削除。

入力ソースで何故かUSが入ってきて消せない場合は以下を参照。

  • Macで入力ソースにいつの間にかU.S.が入ってきて消せない時の対処法

トラックパッド

  • ポイントとクリック
    • 調べる&データ検出: 3本指でタップ、だけから1本指で強めのクリック、も選択出来る様になっている（これがデフォルト)、とりあえずそのままに。
    • 副ボタン: 2本指でクリック、から右下隅をクリック、に変更。
    • タップでクリップ: チェックが外れているのでチェックを入れる。
    • 軌跡の速さ: 最速に
    • クリック: 中、から、弱い、に。（物理クリックでないのでこれが変更できるのが凄い。)
    • 強めのクリックと感覚フィードバック: この項目を外すと強めのクリック(2段めのクリック)が出来なくなるらしい。出来るのは1本指のみっぽい。
  • スクロールとズーム
    • スクロールの方向: ナチュラルのままで。
    • スマートズーム: チェックを外す。
  • その他のジェスチャ
    • ページ間をスワイプ: チェックを外す。
    • フルスクリーンアプリケーション間をスワイプ: 3本指から４本指で左右にスワイプ、に。（３本指はBTTで前のページ、次のページへ割当）
    • 通知センター: チェックを外す。
    • アプリケーションでExpose: チェックを入れる。3本指から4本指で下にスワイプ、に変更。
    • Launchpad: チェックを外す。
    • デスクトップを表示: チェックを外す。

iCloud

  • 以下からチェックを外す
    • iCloud Drive
    • 写真 (直接外せないので、オプションの中にある項目を全て外す。)
    • メール

共有

  • コンピューター名を適当に変更(前のやつと同じにしようとしたら同じ名前がネットワーク内にあるから駄目、って言われた。設定した後に同じ名前同士のがつながるとどうなるのか?)
  • リモートログイン: チェックを入れる。次のユーザーのみ、にチェックが入っていることを確認して現在のユーザーを加える。(Administratorsは削除した方がよいかも。削除しよう。

日付と時刻

  • 時計
    • 日付を表示

アクセシビリティ

  • マウスとトラックパッド
    • トラックパッドオプション
      • ドラッグを有効にする: チェック
      • ドラッグロックあり、に変更
