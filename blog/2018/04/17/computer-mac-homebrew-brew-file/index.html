BrewfileгҒ§гҒ®з®ЎзҗҶ

HomebrewгҒ§гӮӨгғігӮ№гғҲгғјгғ«гҒ—гҒҹгӮҠгҒҷгӮӢгғ‘гғғгӮұгғјгӮёгҒ«й–ўгҒ—гҒҰгҒҜ Brew-fileгӮ’дҪҝгҒЈгҒҰBrewfileгҒ§гғӘгӮ№гғҲгӮ’з®ЎзҗҶгҒ—гҒҰгҒ„гҒҫгҒҷгҖӮ

гҒ“гӮҢгҒ®гӮӨгғігӮ№гғҲгғјгғ«гҒҜ

$ brew install rcmdnk/file/brew-file

гҒ§еҮәжқҘгҒҫгҒҷгҖӮ гӮігғһгғігғүгҒҜbrew file <subcommand>гҒ«гҒӘгӮҠгҒҫгҒҷгҖӮ

дёҖж–№HomebrewгҒ«гҒҜдёҖеҝңе…¬ејҸзҡ„гҒӘbundleгҒЁгҒ„гҒҶгӮігғһгғігғүгӮӮгҒӮгӮҠгҒҫгҒҷгҖӮ

гҒ©гҒЎгӮүгӮӮHomebrewй–ўйҖЈ(brew or cask, + Apple StoreгӮўгғ—гғӘгӮ’masгҒ§з®ЎзҗҶ)гҒ§ гҒ„гӮҢгҒҹгғ‘гғғгӮұгғјгӮё/гӮўгғ—гғӘгӮ’BrewfileгҒ«жӣёгҒҚеҮәгҒ—гҒҹгӮҠгҖҒ BrewfileгӮ’гӮӮгҒЁгҒ«гҒҫгҒЁгӮҒгҒҰгӮӨгғігӮ№гғҲгғјгғ«гҒ—гҒҹгӮҠгҒҷгӮӢгӮігғһгғігғүгҒ§гҒҷгҖӮ

Brew-fileгҒ§гҒҜеҠ гҒҲгҒҰ

  • brew installгӮ„brew uninstallжҷӮгҒ«иҮӘеӢ•зҡ„гҒ«BrewfileгӮ’гӮўгғғгғ—гғҮгғјгғҲгҒҷгӮӢгҖӮ
  • GitHubгҒ®гғ¬гғқгӮёгғҲгғӘгҒӘгҒ©гӮ’дҪҝгҒЈгҒҰз®ЎзҗҶгҒҢеҮәжқҘгӮӢгҖӮд»–гҒ®гғһгӮ·гғігҒЁгҒ®е…ұз”ЁгҒҢгӮ„гӮҠгӮ„гҒҷгҒ„(dotfilesзҡ„гҒӘ)гҖӮ
  • иӨҮж•°гҒ®гғ•гӮЎгӮӨгғ«еҪўејҸгҒҢгҒӮгӮӢ(bundleгҒ®еҮәеҠӣгӮӮиӘӯгҒҝиҫјгӮҒгӮӢ)гҖӮ

еҚҳгҒ«гғ•гӮЎгӮӨгғ«гҒ«гғӘгӮ№гғҲгӮ’жӣёгҒҚеҮәгҒ—гҒҰгҒқгӮҢгӮ’гӮӮгҒЁгҒ«гӮӨгғігӮ№гғҲгғјгғ«гҒҷгӮӢгҒ гҒ‘гҒӘгӮүгҖҒ brew listгҒЁгҒӢгҒЁз°ЎеҚҳгҒӘгӮ·гӮ§гғ«гӮ№гӮҜгғӘгғ—гғҲгҒ§гӮӮеҮәжқҘгҒҰгҒ—гҒҫгҒҶгҒ“гҒЁгҒ§гҒҷгҒҢгҖҒ гҒ“гҒ®иҫәгҒ®иҮӘеӢ•гҒ§з®ЎзҗҶгҒҷгӮӢгҒ“гҒЁгҒҢгҒ—гҒҹгҒҸгҒҰBrew-fileгӮ’дҪҝгҒЈгҒҰгҒ„гҒҫгҒҷгҖӮ

brewгғ‘гғғгӮұгғјгӮёгҒ®ж•ҙзҗҶ

гҒ“гӮҢгҒҢдёҖз•ӘгӮ„гҒЈгҒӢгҒ„гҒ§гҒҷгҒҢгҖҒHomebrewгҒ§гҒҜ д»–гҒ®гғ‘гғғгӮұгғјгӮёгҒ«дҫқеӯҳгҒ—гҒҰгҒ„гӮӢгӮӮгҒ®гӮ’гӮӨгғігӮ№гғҲгғјгғ«гҒҷгӮӢйҡӣгҒ«гҒҜ дҫқеӯҳе…ҲгҒ®гғ‘гғғгӮұгғјгӮёгҒҢе…ҘгҒЈгҒҰгҒ„гҒӘгҒ‘гӮҢгҒ°иҮӘеӢ•зҡ„гҒ«гӮӨгғігӮ№гғҲгғјгғ«гҒ•гӮҢгҒҫгҒҷгҖӮ

е…ғгҒ®гӮўгғігӮӨгғігӮ№гғҲгғјгғ«гҒҷгӮӢйҡӣгҒ«гҒҜгҒқгӮҢгӮүгҒ®гғ‘гғғгӮұгғјгӮёгҒҜгӮўгғігӮӨгғігӮ№гғҲгғјгғ«гҒ•гӮҢгҒӘгҒ„гҒ®гҒ§ й•·гҒ„гҒ“гҒЁдҪҝгҒЈгҒҰгӮӢгҒЁдҪҝгӮҸгҒӘгҒ„гҒ®гҒ«е…ҘгҒЈгҒҰгӮӢгғ‘гғғгӮұгғјгӮёгҒЁгҒӢгӮӮзөҗж§ӢгҒӮгӮӢгҒҜгҒҡгҒ§гҒҷгҖӮ

гӮӨгғігӮ№гғҲгғјгғ«гҒ—гҒҹжҷӮгҒ«зӣҙжҺҘе…ҘгӮҢгҒҹгҒ®гҒӢдҪ•гҒӢгҒ®дҫқеӯҳгғ‘гғғгӮұгғјгӮёгҒЁгҒ—гҒҰе…ҘгҒЈгҒҹгҒ®гҒӢгҒҜ еҺ»е№ҙгҖҒHomebrewе°Һе…ҘгҒ•гӮҢгҒҹ installed_on_request/installed_as_dependency гҒЁгҒ„гҒҶжғ…е ұгҒ§иҰӢгӮӢгҒ“гҒЁгҒҢеҮәжқҘгҒҫгҒҷгҖӮ

  • HomebrewгҒ§зӣҙжҺҘгӮӨгғігӮ№гғҲгғјгғ«гҒ—гҒҹгҒӢдҫқеӯҳй–ўдҝӮгҒ§гӮӨгғігӮ№гғҲгғјгғ«гҒ•гӮҢгҒҹгҒӢзўәиӘҚеҮәжқҘгӮӢж§ҳгҒ«гҒӘгҒЈгҒҹ

гҒҹгҒ гҖҒгҒ“гҒ®жҢҮжЁҷгҒҢе°Һе…ҘгҒ•гӮҢгӮӢеүҚгҒ«гӮӨгғігӮ№гғҲгғјгғ«гҒ•гӮҢгҒҹгғ‘гғғгӮұгғјгӮёгҒ«й–ўгҒ—гҒҰгҒҜ гҒ“гӮҢгӮүгҒ®еҖӨгҒҢtrueгҒЁгҒӢfalseгҒ§гҒӘгҒҸгҒҰnullгҒ«гҒӘгҒЈгҒҰгӮӢгҒ®гҒ§ еҲҶгҒӢгӮүгҒӘгҒ„е ҙеҗҲгӮӮгҒӮгӮҠгҒҫгҒҷгҖӮ

гҒҫгҒҹгҖҒгҒҹгҒЁгҒҲгҒ°pythonгҒЁгҒӢгҒҜжҷ®йҖҡгҒ«е…ҘгӮҢгҒҹгҒ„гӮӮгҒ®гҒ гҒЈгҒҹгӮҠгҒҷгӮӢгҒӢгӮӮгҒ—гӮҢгҒҫгҒӣгӮ“гҒҢгҖҒ иҮӘеҲҶгҒ§е…ҘгӮҢгӮӢгӮҲгӮҠгӮӮе…ҲгҒ«д»–гҒ®гғ‘гғғгӮұгғјгӮёгҒ®дҫқеӯҳгҒЁгҒ—гҒҰе…ҘгҒЈгҒҰгҒ„гӮӢгҒЁ installed_on_requestгҒҢfalseгҒ«гҒӘгҒЈгҒҰгӮӢгҒ“гҒЁгӮӮгҒӮгӮҠгҒҫгҒҷгҖӮ

гҒӘгҒ®гҒ§гҖҒдҪҝгҒҶгҒЁгҒ—гҒҹгӮүиҰӢиҰҡгҒҲгҒ®гҒӘгҒ„гғ‘гғғгӮұгғјгӮёгӮ’иӘҝгҒ№гҒҰinstalled_as_dependencyгҒҢtrueгҒӢгҒ©гҒҶгҒӢиҰӢгӮӢгҖҒгҒЁиЁҖгҒЈгҒҹж„ҹгҒҳгҒ§гҒ—гӮҮгҒҶгҒӢгҖӮ

гҒҫгҒҹгҖҒcaskгӮӮгҒӘгҒҸгҖҒApp StoreгӮ„гӮ·гӮ№гғҶгғ гӮўгғ—гғӘгҒ§гӮӮз„ЎгҒ„гӮӮгҒ®гҒ§зӣҙжҺҘгӮӨгғігӮ№гғҲгғјгғ«гҒ—гҒҹгӮӮгҒ®гҒҢгҒӮгӮҢгҒ° caskгҒ®иҮӘдҪңгӮӮеҸҜиғҪгҒӘйҷҗгӮҠиЎҢгҒ„гҒҫгҒҷгҖӮ

  • Homebrew CaskгҒ®гӮўгғғгғ—гғҮгғјгғҲзӯү

ж•ҙзҗҶгҒ—гҒҰгҒ„гӮӢжҷӮгҒ«гҖҒbrew uninstallгӮ’иЎҢгҒЈгҒҰгӮӮbrew cask listгҒ«ж®ӢгӮҠз¶ҡгҒ‘гӮӢ гӮўгғ—гғӘгҒҢеҮәгҒҰгҒҸгӮӢдәӢгҒҢгҒӮгӮҠгҒҫгҒҷгҒҢгҖҒгҒ“гӮҢгҒҜ /usr/local/Caskroom/гҒ«гӮўгғ—гғӘеҗҚгҒ®гғҮгӮЈгғ¬гӮҜгғҲгғӘгҒҢж®ӢгҒЈгҒҰгҒ„гҒҰ дёӯиә«гҒҜгҒҷгҒ§гҒ«еүҠйҷӨгҒ•гӮҢгҒҰгӮӢж§ҳгҒӘзҠ¶ж…ӢгҒ§гҒҷгҖӮ(дёҠгҒ®иЁҳдәӢеҸӮз…§гҖӮ)

/usr/local/Caskroom/гҒ«гҒӮгӮӢгғҮгӮЈгғ¬гӮҜгғҲгғӘгӮ’иӘҝгҒ№гҒҰдёӯиә«гҒҢз„ЎгҒ„гғҮгӮЈгғ¬гӮҜгғҲгғӘгҒҜ дҪ•гӮүгҒӢгҒ®дёҚе…·еҗҲгҒ§ж®ӢгҒЈгҒҰгҒ—гҒҫгҒЈгҒҹгӮҙгғҹгҒӘгҒ®гҒ§зӣҙжҺҘгғҮгӮЈгғ¬гӮҜгғҲгғӘгӮ’еүҠйҷӨгҒ—гҒҰгҒ—гҒҫгҒҷгҖӮ

гҒ©гҒҶгӮӮCaskгҒ®еҗҚеүҚгҒҢеӨүгӮҸгҒЈгҒҹгӮҠд»–гҒ®гӮўгғ—гғӘгҒЁгҒҫгҒЁгӮҒгҒҰгӮӨгғігӮ№гғҲгғјгғ«гҒ•гӮҢгӮӢж§ҳгҒ«гҒӘгҒЈгҒҹгӮҠгҒ—гҒҹйҡӣгҒ« е…ғгҒ®CaskгҒ®гғҮгӮЈгғ¬гӮҜгғҲгғӘгҒҢз©әгҒ®гҒҫгҒҫж®ӢгҒЈгҒҰгҒ—гҒҫгҒЈгҒҹгӮҠгҒҷгӮӢгӮҲгҒҶгҒ§гҒҷгҖӮ
