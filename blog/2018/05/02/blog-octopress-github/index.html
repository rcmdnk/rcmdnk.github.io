OctopressгӮ’дҪҝгҒҶгҒ“гҒЁгҒ§жҷ®ж®өдҪҝгҒЈгҒҰгҒӘгҒӢгҒЈгҒҹRubyгӮӮгҒӮгӮӢзЁӢеәҰгҒҜдҪҝгҒҲгӮӢж§ҳгҒ«гҒӘгҒЈгҒҹгҒ—гҖҒ HugoгӮ’дҪҝгҒ„гҒӘгҒҢгӮүGoгӮ’еӢүеј·гҒ§гҒҚгӮӢгҒ®гӮӮиүҜгҒ„гҒӘгҖҒгҒЁгҒ„гҒҶзӮ№гӮӮгҖӮ

гҒҹгҒ зӢ¬иҮӘгғ—гғ©гӮ°гӮӨгғігҒЁгҒӢиүІгҖ…гҒЁжүӢгӮ’е…ҘгӮҢгҒҰгӮӢгҒ®гҒ§гҖҒ з”Ёж„ҸгҒ•гӮҢгҒҰгҒ„гӮӢеӨүжҸӣгӮігғјгғүгҒЁгҒӢгҒ гҒЁе•ҸйЎҢгҒҢеӨҡзҷәгҒ—гҒҰгҒ—гҒҫгҒҶгҒ®гҒ§ еӨүжҸӣгғ«гғјгғ«гӮӮгҒӮгӮӢзЁӢеәҰиҮӘеҲҶгҒ§дҪңгӮүгҒӘгҒ„гҒЁгҒ„гҒ‘гҒӘгҒ„гҖҒгҒЁгҒ„гҒҶгҒ“гҒЁгҒ§гҒҫгҒ еҮәжқҘгҒҰгҒҫгҒӣгӮ“гҖӮ

гҒ“гҒ®еҲ¶йҷҗгҒҢгҒӢгҒӢгҒЈгҒҰгӮӮгҒҷгҒҗгҒ«гӮөгӮӨгғҲгҒҢиҰӢгӮҢгҒӘгҒҸгҒӘгӮӢгӮҸгҒ‘гҒ§гҒҜгҒӘгҒҸгҒҰ гҒҫгҒҡгҒҜгӮөгғқгғјгғҲгҒӢгӮүйҖЈзөЎгҒҢжқҘгӮӢгҖҒгҒЁгҒӮгӮӢгҒ®гҒ§и¶…гҒҲгӮӢгҒӢи¶…гҒҲгҒӘгҒ„гҒӢгҖҒ гҒЁгҒ„гҒҶе ҙеҗҲгҒ«гҒҜгҒҷгҒҗгҒ«еҝғй…ҚгҒҷгӮӢеҝ…иҰҒгҒҜгҒӘгҒ„гҒЁжҖқгҒ„гҒҫгҒҷгҖӮ

гҒ§гҒҷгҒҢгӮігғігӮ№гӮҝгғігғҲгҒ«и¶…гҒҲе§ӢгӮҒгҒҹгӮүиҖғгҒҲгҒӘгҒ„гҒЁгҒ„гҒ‘гҒҫгҒӣгӮ“гҖӮ

гҒҫгҒҹгҖҒгғ¬гғқгӮёгғҲгғӘгҒ®еҲ¶йҷҗгҒЁгҒ—гҒҰ1GBзЁӢеәҰгҒЁгҒ„гҒҶгҒ“гҒЁгҒӘгҒ®гҒ§гҒҷгҒҢгҖҒ гҒ“гҒ®гӮөгӮӨгғҲгҒҜгҒҫгҒ 700MBдҪҚгҒ§гҒҷгҒҢжңҖиҝ‘еӨ§гҒҚгҒӘз”»еғҸгғ•гӮЎгӮӨгғ«(80MBдҪҚ)гӮ’ йҖҒгӮҚгҒҶгҒЁгҒ—гҒҹгӮүеӨ§гҒҚгҒҷгҒҺгҒҰй§„зӣ®гҖҒгҒЁиЁҖгӮҸгӮҢгҒҫгҒ—гҒҹгҖӮ (git pushжҷӮгҒ«гӮЁгғ©гғјгҒ«гҒӘгӮӢгҖӮ)

еҗҢгҒҳгғ•гӮЎгӮӨгғ«гӮ’д»–гҒ®гғ¬гғқгӮёгғҲгғӘгҒ«йҖҒгҒЈгҒҹгҒЁгҒ“гӮҚе•ҸйЎҢгҒӘгҒҸйҖҒгӮҢгҒҫгҒ—гҒҹгҖӮ

дёҖеҝңGitHubзҡ„гҒ«гҒҜ100MBгӮ’зӣ®е®үгҒ«дёҖгҒӨгҒ®гғ•гӮЎгӮӨгғ«гҒ®гӮөгӮӨгӮәгҒ®еҲ¶йҷҗгҒҢгҒӮгӮҠгҒҫгҒҷгҖӮ

What is my disk quota? - User Documentation

гҒ§гҒҷгҒҢдёҖгҒӨгҒ®гғ¬гғқгӮёгғҲгғӘгҒ§гҒӮгӮӢзЁӢеәҰдҪҝгҒЈгҒҰгҒ„гҒҸгҒЁ дёҖгҒӨгҒ®гғ•гӮЎгӮӨгғ«гӮөгӮӨгӮәгҒ®еҲ¶йҷҗгӮӮгҒҚгҒӨгҒҸгҒӘгҒЈгҒҰгҒ„гҒҸгӮҲгҒҶгҒ§гҒҷгҖӮ

гҒҫгҒӮгҖҒ80MBгҒ®гғ•гӮЎгӮӨгғ«гҒҜгҒЎгӮҮгҒЈгҒЁгҒ©гҒҶгҒ«гҒӢгҒ—гҒӘгҒҚгӮғгҒ„гҒ‘гҒӘгҒ„гғ¬гғҷгғ«гҒ§гҒҜгҒӮгӮӢгҒ®гҒ§гҒҷгҒҢгҖӮ

гғ¬гғқгӮёгғҲгғӘгҒ®еҲ¶йҷҗгӮӮгҒӮгӮҢгҒ§гҒҷгҒҢгҖҒгҒқгӮҢд»ҘдёҠгҒ«еёҜеҹҹеҲ¶йҷҗгҒҢгҒӮгӮӢгҒ®гҒ§ д»–гҒ«з§»гӮҠгҒҹгҒ„гҖҒгҒЁгҒ„гҒҶдәәгҒҢзөҗж§Ӣеұ…гӮӢгӮҲгҒҶгҒ§гҒҷгҖӮ

гҒқгӮ“гҒӘдёӯгҒ§жңҖиҝ‘гҖҒNetlifyгҒЁгҒ„гҒҶгӮөгғјгғ“гӮ№гҒҢгҒЎгӮҮгҒЈгҒЁжөҒиЎҢгҒЈгҒҰгӮӢжЁЎж§ҳгҒ§гҖҒ GitHub PagesгҒӢгӮүз§»гҒЈгҒҹгҖҒгҒЁгҒ„гҒҶдәәгҒҢзөҗж§ӢиҰӢгӮүгӮҢгҒҫгҒ—гҒҹгҖӮ

Netlify: All-in-one platform for automating modern web projects.

Netlifyз§»иЎҢиЁҳпјҡGitHub PagesгҒӢгӮүзӢ¬иҮӘгғүгғЎгӮӨгғіпјӢNetlifyгҒ®еј•гҒЈи¶ҠгҒ—гҒ§гӮ„гҒЈгҒҹгҒ“гҒЁ TPB

NetlifyгҒҜйқҷзҡ„гӮөгӮӨгғҲгӮ’з°ЎеҚҳгҒ«е…¬й–ӢгҒ§гҒҚгӮӢгӮөгғјгғ“гӮ№гҒ§гҖҒ GitHubгҒЁйҖЈжҗәгҒ—гҒҰCIгӮ’NetlifyеҒҙгҒ§еӢ•гҒӢгҒ—гҒҰгӮөгӮӨгғҲгӮ’гғҮгғ—гғӯгӮӨгҒ—гҒҹгӮҠгҒҷгӮӢгҒ“гҒЁгҒҢеҮәжқҘгҒҫгҒҷгҖӮ

гҒҹгҒ гҖҒгҒ“гҒ®CIгҒҜ15еҲҶгҒ®еҲ¶йҷҗгҒҢгҒӮгӮӢгӮүгҒ—гҒҸгҖҒOctopressгҒ§й•·гҒҸгҒӢгҒӢгӮӢе ҙеҗҲгҒ«гҒҜгҒЎгӮҮгҒЈгҒЁеҺігҒ—гҒ„гҒӢгӮӮгҖӮ (гҒ“гҒ®гӮөгӮӨгғҲгҒӘгӮүеӨҡеҲҶеӨ§дёҲеӨ«гҖӮ)

гҒҫгҒҹгҖҒеӨ–гҒӢгӮүгғ•гӮЎгӮӨгғ«гӮ’зӣҙжҺҘйҖҒгӮҚгҒҶгҒЁгҒ—гҒҹе ҙеҗҲгҖҒAPIгӮ’дҪҝгҒЈгҒҰйҖҒгҒЈгҒҹгӮҠгҒҷгӮӢгҒЁ200гғӘгӮҜгӮЁгӮ№гғҲ/еҲҶгҒ®еҲ¶йҷҗгҒҢгҒӮгӮӢгҒ®гҒ§ ZipгҒЁгҒӢгҒ§гҒҫгҒЁгӮҒгҒҰйҖҒгҒЈгҒҹгӮҠгҒҷгӮӢгҒ“гҒЁгӮ’иҖғгҒҲгҒӘгҒ„гҒЁгҒ„гҒ‘гҒӘгҒ„гҒЁгҒ®гҒ“гҒЁгҖӮ

гҒқгӮ“гҒӘгҒ“гӮ“гҒӘгҒ§гӮӮз§»гҒЈгҒҰдәәгҒҢеұ…гӮӢгҒ®гҒҜеёҜеҹҹеҲ¶йҷҗгҒҢGitHub PagesгӮҲгӮҠгӮӮеӨ§гҒҚгҒ„гҒӢгӮүгҒЁгҒ„гҒҶгҒ“гҒЁгҒ§гҒҷгҒҢгҖҒ еҲ¶йҷҗгҒЁгҒ—гҒҰгҒҜ100GB per monthгҒ§гҒҷгҖӮ

гҒҹгҒ гҖҒGitHub PagesгҒЁйҮҸзҡ„гҒ«гҒҜеҗҢгҒҳгҒ§гҖҒGitHub PagesеҒҙгҒ« 100,000 requests per monthгҒҢз„ЎгҒҸгҒӘгҒЈгҒҹгҒ®гҒ§е®ҹгҒҜзҸҫеңЁгҒҜе®ҹиіӘзҡ„гҒ«дёҖз·’гҒӘгҒ®гҒӢгӮӮгҒ—гӮҢгҒҫгҒӣгӮ“гҖӮ

гҒқгҒҶиҖғгҒҲгӮӢгҒЁгҒӮгҒҫгӮҠж„Ҹе‘ігҒӘгҒ„гҒӢгӮӮгҖӮ

гҒӮгҒЁгҖҒгҒ„гҒҡгӮҢгҒ«гҒ—гӮҚгҖҒзҸҫеңЁгҒҜCloudflareгӮ’дҪҝгҒЈгҒҰCDNгӮ’йҖҡгҒ—гҒҰй…ҚдҝЎгҒ—гҒҰгҒ„гӮӢгҒ®гҒ§ гҒқгӮӮгҒқгӮӮгҒ“гҒ®иҫәгҒ®иЁҲз®—гҒҢгҒӮгҒҫгӮҠж„Ҹе‘ігӮ’гҒӘгҒ—гҒҰгҒҫгҒӣгӮ“гҖӮ

гҒӘгҒ®гҒ§NetlifyгҒҜгҒЎгӮҮгҒЈгҒЁж°—гҒ«гҒӘгҒЈгҒҹгҒ®гҒ§гҒҷгҒҢ CloudflareгӮ’дҪҝгҒЈгҒҰгӮӢд»ҘдёҠгҒӮгҒҫгӮҠеёҜеҹҹиҮӘдҪ“гҒҜж°—гҒ«гҒ—гҒӘгҒҸгҒҰгӮӮиүҜгҒ„гҒЁгҒ“гӮҚгҒ гҒЁжҖқгҒ„гҒҫгҒҷгҖӮ

гӮ«гӮ№гӮҝгғ гғүгғЎгӮӨгғігҒ®HTTPSеҢ–гҒҢгӮөгғқгғјгғҲгҒ•гӮҢгҒҹгҒ®гҒ§GitHub PagesгҒ гҒ‘гҒ§й…ҚдҝЎгҒ—гӮҲгҒҶгҒЁжҖқгҒҶгҒЁ еӨҡе°‘ж°—гҒ«гҒӘгӮӢгҒЁгҒ“гӮҚгҒ§гҒҷгҒҢгҖҒеҲ¶йҷҗзҡ„гҒ«NetlifyгҒ®ж–№гҒҢе„ӘгӮҢгҒҰгҒ„гӮӢгҖҒгҒЁгҒ„гҒҶгҒ“гҒЁгӮӮд»ҠгҒҜз„ЎгҒ„гӮҲгҒҶгҒ§гҒҷгҖӮ

гҒЁгҒ„гҒҶгҒ“гҒЁгҒ§гҒ“гҒЎгӮүгҒҜгҒЁгӮҠгҒӮгҒҲгҒҡеӨүжӣҙгҒҷгӮӢж„Ҹе‘ігҒҜз„ЎгҒ„гҒӢгҒӘгҖҒгҒЁгҒ„гҒЈгҒҹж„ҹгҒҳгҒ§гҒҷгҖӮ
