PythonгҒ®гғ‘гғғгӮұгғјгӮёгҒ§ruamel.yamlгҒҢеҝ…иҰҒгҒӘгҒ®гҒ§е…ҘгҒЈгҒҰгҒ„гҒӘгҒ„е ҙеҗҲгҒҜ

$ pip install ruamel.yaml

гҒӘгҒ©гҒ§гӮӨгғігӮ№гғҲгғјгғ«гҒ—гҒҰгҒҸгҒ гҒ•гҒ„гҖӮ

XML to YAML

YAMLгҒ§зӣҙжҺҘгғ•гӮЈгғ«гӮҝгӮ’жӣёгҒ„гҒҰгӮӮиүҜгҒ„гҒ§гҒҷгҒҢгҖҒ гҒҷгҒ§гҒ«GmailгҒ§гғ•гӮЈгғ«гӮҝгӮ’иЁӯе®ҡгҒ—гҒҰгҒ„гҒҹгӮҠгҒҷгӮӢе ҙеҗҲгҒқгӮҢгӮ’гӮЁгӮӯгӮ№гғқгғјгғҲгҒ—гҒҰ YAMLгҒ«еӨүжҸӣгҒ—гҒҰз·ЁйӣҶгҒҷгӮӢгҒ“гҒЁгҒҢеҮәжқҘгҒҫгҒҷгҖӮ

гҒҫгҒҡгҖҒGmailгҒӢгӮүгғ•гӮЈгғ«гӮҝгӮ’XMLеҪўејҸгҒ§гӮЁгӮӯгӮ№гғқгғјгғҲгҒ—гҒҫгҒҷгҖӮ гӮЁгӮӯгӮ№гғқгғјгғҲгҒҜGmailгҒ®иЁӯе®ҡгҒӢгӮүгғ•гӮЈгғ«гӮҝгҒЁгғ–гғӯгғғгӮҜдёӯгҒ®гӮўгғүгғ¬гӮ№гҒ®гғҡгғјгӮёгҒ«иЎҢгҒҚгҖҒ еҮәеҠӣгҒ—гҒҹгҒ„гғ•гӮЈгғ«гӮҝгӮ’гғҒгӮ§гғғгӮҜгҒ—гҒҰдёӢгҒ«гҒӮгӮӢгӮЁгӮӯгӮ№гғқгғјгғҲгғңгӮҝгғігӮ’жҠјгҒ—гҒҫгҒҷгҖӮ

йҖҡеёёгҒҜдёҖиҰ§гҒ®дёӢгҒ«гҒӮгӮӢгҒҷгҒ№гҒҰгӮ’жҠјгҒ—гҒҰгҒҷгҒ№гҒҰйҒёжҠһгҒ—гҒҰгӮЁгӮӯгӮ№гғқгғјгғҲгҒҷгӮҢгҒ°иүҜгҒ„гҒЁжҖқгҒ„гҒҫгҒҷгҖӮ

гғЎгғјгғ«гҒ®гғ•гӮЈгғ«гӮҝгғ«гғјгғ«гҒ®дҪңжҲҗ - Gmail гғҳгғ«гғ—

гӮЁгӮӯгӮ№гғқгғјгғҲгҒ—гҒҹmailFilters.xmlгҒҜд»ҘдёӢгҒ®ж§ҳгҒӘеҪўгҒ«гҒӘгҒЈгҒҰгӮӢгҒЁжҖқгҒ„гҒҫгҒҷгҖӮ

гҒ“гӮҢгӮ’еҶҚгҒігӮӨгғігғқгғјгғҲгҒҷгӮҢгҒ°гғ•гӮЈгғ«гӮҝгҒҢдҪңжҲҗгҒ§гҒҚгӮӢгҒ®гҒ§ гғҗгғғгӮҜгӮўгғғгғ—гҒЁгҒ—гҒҰеҸ–гҒЈгҒҰгҒҠгҒ„гҒҹгӮҠеҮәжқҘгҒҫгҒҷгҖӮ

д»ҘеүҚгҒҜгҒ“гӮҢгӮ’зӣҙжҺҘз·ЁйӣҶгҒ—гҒҰгҒ„гҒҫгҒ—гҒҹгҒҢгҒ„гӮҚгҒ„гӮҚйқўеҖ’гҒӘгҒ®гҒ§YAMLеҪўејҸгҒ§з®ЎзҗҶгҒ—гӮҲгҒҶгҖҒгҒЁгҒ„гҒҶгҒ“гҒЁгҒ§ gfm_extracгӮ’дҪҝгҒ„гҒҫгҒҷгҖӮ

gfm_extractгҒҜз¬¬дёҖеј•ж•°гҒҢгӮӨгғігғ—гғғгғҲгғ•гӮЎгӮӨгғ«гҖҒз¬¬дәҢеј•ж•°гҒҢгӮўгӮҰгғҲгғ—гғғгғҲгғ•гӮЎгӮӨгғ«гҒ®еҗҚеүҚгҒ§гҖҒ жҢҮе®ҡгҒҢгҒӘгҒ‘гӮҢгҒ°гӮӨгғігғ—гғғгғҲгҒ«mailFilters.xmlгҖҒгӮўгӮҰгғҲгғ—гғғгғҲгҒ«mailFilters.yamlгҒҢжҢҮе®ҡгҒ•гӮҢгҒҫгҒҷгҖӮ

$ gfm_extract

гҒ§еҮәжқҘгҒҹmailFilters.yamlгӮ’иҰӢгҒҰгҒҝгӮӢгҒЁ

гҒ“гӮ“гҒӘж„ҹгҒҳгҒ®гӮ·гғігғ—гғ«гҒӘгӮӮгҒ®гҒ«гҒӘгҒЈгҒҰгҒ„гӮӢгҒӢгҒЁжҖқгҒ„гҒҫгҒҷгҖӮ

гӮӨгғігғқгғјгғҲгҒҷгӮӢйҡӣгҖҒ дёҠгҒ®XMLгғ•гӮЎгӮӨгғ«гҒ®дёӯгҒ§е®ҹйҡӣгҒ«еҝ…иҰҒгҒӘйғЁеҲҶгҒ гҒ‘гӮ’жҠңгҒҚеҮәгҒ—гҒҹеҪўгҒ§гҒҷгҖӮ

еҹәжң¬зҡ„гҒ«entryгӮҝгӮ°гҒҢдёҖгҒӨгҒ®гғ•гӮЈгғ«гӮҝгӮ’иЎЁгҒ—гҒҰгҒ„гҒҰгҖҒгҒқгҒ®дёӯгҒ® apps:propertyгӮҝгӮ°гҒҢгғ•гӮЈгғ«гӮҝжқЎд»¶гҖҒеҮҰзҪ®ж–№жі•гҒ«гҒӨгҒ„гҒҰгҒ®еҶ…е®№гҒ«гҒӘгҒЈгҒҰгҒ„гҒҰгҖҒ гҒқгӮҢд»ҘеӨ–гҒҜеҝ…иҰҒгҒӮгӮҠгҒҫгҒӣгӮ“гҖӮ

namespacesгҒ«й–ўгҒ—гҒҰгҒҜдҫҝе®ңдёҠж®ӢгҒ—гҒҰгҒ„гҒҫгҒҷгҒҢгҖҒ gfm_makeгҒҷгӮӢгҒЁгҒҚгҒ«гҒ“гӮҢгӮүгӮ’иЈңе®ҢгҒҷгӮӢгӮҲгҒҶгҒ«гҒ—гҒҰгҒӮгӮӢгҒ®гҒ§ иҮӘдҪңгҒҷгӮӢйҡӣгҒ«гҒҜгҒӘгҒҸгҒҰгӮӮеӨ§дёҲеӨ«гҒ§гҒҷгҖӮ

YAMLгҒ§гғ•гӮЈгғ«гӮҝгӮ’дҪңгӮӢ

з°ЎеҚҳгҒ«дҪңгӮҠж–№гӮ’зҹҘгӮӢж–№жі•гҒЁгҒ—гҒҰгҒҜгҖҒ йҒ©еҪ“гҒӘгғ•гӮЈгғ«гӮҝгӮ’GmailгҒ§дҪңгҒЈгҒҰ гҒқгӮҢгӮүгӮ’гӮЁгӮӯгӮ№гғқгғјгғҲгҒ—гҖҒгҒқгҒ®еҶ…е®№гӮ’иҰӢгҒҰгҒҝгӮӢгҒ®гҒҢиүҜгҒ„гҒ§гҒҷгҖӮ gfm_extracгҒ§YAMLгҒ«еӨүжӣҙгҒҷгӮӢгҒ“гҒЁгҒ§жӣҙгҒ«гӮҸгҒӢгӮҠгӮ„гҒҷгҒҸгҒӘгӮӢгҒҜгҒҡгҒ§гҒҷгҖӮ

YAMLгҒ®еҪўејҸгҒЁгҒ—гҒҰгҒҜгҖҒfiltersгҒЁгҒ„гҒҶгӮӯгғјгҒ«еҜҫгҒ—гҒҰ гғ•гӮЈгғ«гӮҝгӮ’й…ҚеҲ—гҒ§й…ҚзҪ®гҒ—гҒҰгҒ„гҒҚгҒҫгҒҷгҖӮ

еҗ„гғ•гӮЈгғ«гӮҝгҒ®дёӯиә«гҒҜй…ҚеҲ—гҒ§гҖҒдёҖгҒӨгҒ«гҒӨгҒҚдёҖгҒӨгҖҒapps:propertyгӮҝгӮ°гҒ«еҜҫеҝңгҒҷгӮӢгӮӮгҒ®гӮ’жӣёгҒ„гҒҰгҒ„гҒҚгҒҫгҒҷгҖӮ гӮӯгғјгҒЁгҒ—гҒҰXMLгҒ®nameеұһжҖ§гӮ’гҖҒеҖӨгҒЁгҒ—гҒҰvalueеұһжҖ§гӮ’жӣёгҒҚгҒҫгҒҷгҖӮ

гғ•гӮЈгғ«гӮҝгҒ®жқЎд»¶гӮӮгғ•гӮЈгғ«гӮҝйҒ©з”ЁеҫҢгҒ®еҮҰзҪ®гҒ«гҒӨгҒ„гҒҰгӮӮ еҗҢгҒҳж§ҳгҒӘиЁҳиҝ°гҒ§еҗҢеҲ—гҒ«жӣёгҒ„гҒҰгҒ„гҒҸеҪўгҒ«гҒӘгӮҠгҒҫгҒҷгҖӮ (XMLгҒ§гӮӮе…ЁгҒҰapps:propertyгҒ§еҗҢж§ҳгҒ®жӣёгҒҚж–№гҒ«гҒӘгҒЈгҒҰгҒ„гҒҰеҢәеҲҘгҒ•гӮҢгҒҰгҒҫгҒӣгӮ“гҖӮ)

гғ•гӮЈгғ«гӮҝгҒ®жқЎд»¶гҒ«дҪҝгҒҲгӮӢгӮӮгҒ®

гғ•гӮЈгғ«гӮҝгҒ«дҪҝгҒҲгӮӢгӮӮгҒ®гҒҜд»ҘдёӢгҒ®гӮӮгҒ®гҒ«гҒӘгӮҠгҒҫгҒҷгҖӮ

GmailдёҠгҒ®иЎЁиЁҳ XMLгҒ®nameеұһжҖ§ иӘ¬жҳҺ

To to гғЎгғјгғ«гҒ®йҖҒдҝЎе…Ҳ
д»¶еҗҚ subject гғЎгғјгғ«гҒ®гӮҝгӮӨгғҲгғ«
еҗ«гӮҖ hasTheWord гҒ“гӮҢгӮ’еҗ«гӮҖгғЎгғјгғ«

гӮөгӮӨгӮә size д»ҘдёӢгҒ®2гҒӨгҒЁгҒ®зө„гҒҝеҗҲгӮҸгҒӣгҒ§жҢҮе®ҡгӮөгӮӨгӮәгҒЁжҜ”ијғгҒҷгӮӢ

MB/KB/гғҗгӮӨгғҲ sizeUnit гӮөгӮӨгӮәгҒ«йҒ©з”ЁгҒҷгӮӢеҚҳдҪҚгҖӮs_sb (гғҗгӮӨгғҲ)гҖҒs_skb (KB)гҖҒs_smb (MB)

еҗ«гӮҖгҖҒеҗ«гҒҫгҒӘгҒ„гҒ«дҪҝгҒҲгӮӢгӮӮгҒ®

еҗ«гӮҖгҖҒеҗ«гҒҫгҒӘгҒ„гҒ«гҒӨгҒ„гҒҰгҒҜиӨҮж•°гҒ®CCгӮ’жҢҮе®ҡгҒ—гҒҹгӮҠгҖҒ гғ©гғҷгғ«гӮ’жҢҮе®ҡгҒ—гҒҹгӮҠгҖҒеҚҳгҒ«гғЎгғјгғ«гҒ®жң¬ж–ҮгҒ«еҗ«гҒҫгӮҢгӮӢгҒӢгҒ©гҒҶгҒӢгҒӘгҒ©гӮ’зө„гҒҝеҗҲгӮҸгҒӣгҒҰдҪҝгҒҲгҒҫгҒҷгҖӮ

гҒ“гӮ“гҒӘж„ҹгҒҳгҒ«гҒӘгӮҠгҒҫгҒҷгҖӮ жңҖеҲқгҒ«гӮЁгӮӯгӮ№гғқгғјгғҲгҒ—гҒҹгӮӮгҒ®гҒ«жҜ”гҒ№гӮӢгҒЁгҒҡгҒ„гҒ¶гӮ“гӮ№гғӘгғ гҒ«гҒӘгӮҠгҒҫгҒ—гҒҹгҒҢ гҒ“гӮҢгҒ§е…ЁгҒҸеҗҢгҒҳгғ•гӮЈгғ«гӮҝгӮ’дҪңжҲҗгҒҷгӮӢгҒ“гҒЁгҒҢеҮәжқҘгҒҫгҒҷгҖӮ

гғ•гӮЈгғ«гӮҝгӮ’гӮӨгғігғқгғјгғҲгҒҷгӮӢгҒ«гҒҜ GmailгҒ®иЁӯе®ҡгҒӢгӮүгғ•гӮЈгғ«гӮҝгҒЁгғ–гғӯгғғгӮҜдёӯгҒ®гӮўгғүгғ¬гӮ№гҒ®гғҡгғјгӮёгҒ«иЎҢгҒҚгҖҒ дёӢгҒ®ж–№гҒ«гҒӮгӮӢгғ•гӮЈгғ«гӮҝгӮ’гӮӨгғігғқгғјгғҲгӮ’жҠјгҒ—гҒҰгӮӨгғігғқгғјгғҲз”»йқўгӮ’иЎЁзӨәгҒ—гҖҒ гғ•гӮЎгӮӨгғ«гӮ’йҒёжҠһгҒ§гғ•гӮЎгӮӨгғ«гӮ’йҒёжҠһеҫҢгҖҒгғ•гӮЎгӮӨгғ«гӮ’й–ӢгҒҸгӮ’жҠјгҒҷгҒЁ гғ•гӮЈгғ«гӮҝгҒҢиЎЁзӨәгҒ•гӮҢгӮӢж§ҳгҒ«гҒӘгӮҠгҒҫгҒҷгҖӮ

гҒ“гҒ®жҷӮзӮ№гҒ§гҒҜгҒҫгҒ GmailгҒ«гӮӨгғігғқгғјгғҲгҒ•гӮҢгҒҰгҒҠгӮүгҒҡгҖҒ гғ•гӮЈгғ«гӮҝгҒ«гғҒгӮ§гғғгӮҜгҒҢе…ҘгҒЈгҒҹзҠ¶ж…ӢгҒ§гҒ•гӮүгҒ«дёӢгҒ®гғ•гӮЈгғ«гӮҝгӮ’дҪңжҲҗгӮ’жҠјгҒҷгҒЁе®ҹйҡӣгҒ«гӮӨгғігғқгғјгғҲгҒ•гӮҢгҒҫгҒҷгҖӮ

гҒ“гҒ®гҒЁгҒҚгҖҒеҸігҒ«гҒӮгӮӢж—ўеӯҳгҒ®гғЎгғјгғ«гҒ«ж–°гҒ—гҒ„гғ•гӮЈгғ«гӮҝгӮ’йҒ©з”ЁгҒҷгӮӢгҒ«гғҒгӮ§гғғгӮҜгҒ—гҒҹгҒҸгҒӘгӮҠгҒҫгҒҷгҒҢгҖҒ гҒҷгҒ№гҒҰгҒ®гғЎгғјгғ«гӮ’иҰӢгӮӢгҒ®гҒ§гӮӮгҒ®гҒҷгҒ”гҒ„жҷӮй–“гҒҢгҒӢгҒӢгӮӢгҒҹгӮҒгҖҒ GmailгӮ’дҪҝгҒ„е§ӢгӮҒгҒҰй–“гӮӮгҒӘгҒ„гҖҒгҒЁгҒӢгҒ§з„ЎгҒ„йҷҗгӮҠгғҒгӮ§гғғгӮҜгҒ—гҒӘгҒ„ж–№гҒҢз„ЎйӣЈгҒ§гҒҷгҖӮ

гғҒгӮ§гғғгӮҜгҒ—гҒӘгҒ„зҠ¶ж…ӢгҒ§100еҖӢгҒҸгӮүгҒ„гҒ®гғ•гӮЈгғ«гӮҝгӮ’дҪңгӮӢгҒ®гҒ«1еҲҶдҪҚгҒӢгҒӢгӮҠгҒҫгҒҷгҖӮ

гҒ“гҒ“гҒ§гҖҒдёҖеәҰгӮЁгӮӯгӮ№гғқгғјгғҲгҒ—гҒҰз·ЁйӣҶгҒ—гҒҰеҶҚеәҰгӮӨгғігғқгғјгғҲгҒҷгӮӢгҒ“гҒЁгӮ’иҖғгҒҲгӮӢгҒЁгҖҒ гғ•гӮЎгӮӨгӮҝгӮ’дҪңжҲҗгӮ’гҒҷгӮӢеүҚгҒ«дёҖеәҰеүҚгҒ®гғ•гӮЈгғ«гӮҝгӮ’гҒҷгҒ№гҒҰеүҠйҷӨгҒ—гҒҰгҒҠгҒ„гҒҹж–№гҒҢиүҜгҒ„гҒ§гҒҷгҖӮ
